POCloud

Due to hosting restrictions, not all features are available online.

Front end ( Angular ) :
  • Error interceptor
  • Auth guards
  • Models
  • Services
Back end ( C# ASP.net ) :
  • Entity framework for interfacing with DB
  • Unit of work and repository pattern
  • Login & registration
  • Upload & download files (Files are saved in Base64 as Byte[] in the DB)
  • Implementation of JWT authentication ( including verifying user's token with a DB when attempting to perform priviliged functions - for example changing the password)
  • Database(datacontext) used is MSSQL

Fix Windows Timer Resolution
    Sets the windows timer resolution to 1 ms, down from the default 15,6ms. This makes the system more responsive overall and likely prevents stuttering or microsttuters from happening in the first place.
